## Defend Game Sfx Logic

### Звуки выстрелов:
1. 1_shoot.mp3 - звук выстрела из 1 оружия (стартовое).
2. 2_shoot.mp3 - звук выстрела из 2 оружия (2-й вид оружия для синих врагов).
3. 3_shoot.mp3 - звук выстрела из 3 оружия (сильный выстрел, для тяжёлых врагов).
4. 4_shoot.mp3 - звук выстрела из 4 оружия (особенный выстрел для 4-го вида врагов).

### Основные звуки:

1. damage.mp3 - урон по игроку.
2. countdown.mp3 - отсчёт (играет только 1 раз)
3. spawn.mp3 - появление врагов

### Звуки магазина после окончании волны:

1. buy.mp3 - играет когда хватает на покупкуоружия.
2. unbuy.mp3 - играет когда пытаешся купить оружие на которое НЕ хватает points.
